PARTE 1: Processamento Bsico nas Imagens e Vdeos

(A) Leitura de imagem em arquivo

A janela no mostra a imagem colorida pois o segundo parmetro da funo cv.imread() foi atribudo o valor 0, que representa ausncia de cor no modo de leitura. Caso a atribuio seja cv.IMREAD_COLOR_BGR a imagem ser convertida para os canais RBG (Red-Blue-Green) e a sada ser uma imagem colorida.

Para corrigir isso, basta alterar a linha de leitura da imagem para:

img = cv.imread('messi5.jpg',cv.IMREAD_COLOR_BGR)

(B) Leitura de vdeo em arquivo

A velocidade de exibio do vdeo, ou seja, a velocidade com que os frames so exibidos na tela depende do tempo de espera entre a exibio de cada frame, controlado pela linha time.sleep(1/25.0) (simula um atraso de 40ms).

Para aumentar a velocidade, basta diminuir o tempo de espera atravs de um acrscimo no valor do denomidador. Por exemplo, se quisermos aumentar para 50fps, basta fazer a seguinte alterao:

time.sleep(1/50.0)

Para reduzir a velocidade, basta aumentar o tempo de espera. Por exemplo, se quisermos diminuir para 10fps, basta fazer a seguinte alterao:

time.sleep(1/10.0)

(C) Leitura de imagem de cmera

Para salvar o frame ao pressionar a tecla 'x', basta adicionar o seguinte comando:

if key == ord('x'):
    cv.imwrite('foto1.png', frame)
    print("Frame saved as 'foto1.png'")

(D) Gravao de vdeo da cmera

Para que a sada do vdeo esteja "normal", ou seja, sem a inverso vertical e com uma velocidade adequada, basta remover o comando cv.flip(frame, 0) que inverte a imagem verticalmente e ajustar o valor da varivel fps para um valor mais alto, como 60.0, que uma taxa de quadros comum para vdeos. Assim, o cdigo para exibir e salvar o vdeo ficar assim:

import numpy as np
import cv2 as cv

cap = cv.VideoCapture(2)

# Get current width of frame
width = cap.get(cv.CAP_PROP_FRAME_WIDTH)   # float
# Get current height of frame
height = cap.get(cv.CAP_PROP_FRAME_HEIGHT) # float
# Define Video Frame Rate in fps
fps = 60.0

# Define the codec and create VideoWriter object
fourcc = cv.VideoWriter_fourcc(*'XVID')
out = cv.VideoWriter('saida.avi', fourcc, fps, (int(width),int(height)) )

while cap.isOpened():
    ret, frame = cap.read()
    if not ret:
        print("Can't receive frame (stream end?). Exiting ...")
        break
    # write the flipped frame
    out.write(frame)
    cv.imshow('frame', frame)
    if cv.waitKey(1) == ord('q'):
        break

# Release everything if job is finished
cap.release()
out.release()
cv.destroyAllWindows()
